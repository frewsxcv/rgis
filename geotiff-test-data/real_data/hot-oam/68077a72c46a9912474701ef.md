## Data creation

From

```
https://oin-hotosm-temp.s3.us-east-1.amazonaws.com/68077a72c46a9912474701ee/0/68077a72c46a9912474701ef.tif
```

Cropped with:

```shell
pixi run rio clip "https://oin-hotosm-temp.s3.us-east-1.amazonaws.com/68077a72c46a9912474701ee/0/68077a72c46a9912474701ef.tif" real_data/hot-oam/68077a72c46a9912474701ef.tif --bounds "246690 4309980 246740 4310030" -f COG
```
